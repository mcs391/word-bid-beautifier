# campus-culture-proposal

校园文化全案建设方案 HTML 生成工具。

## 功能

根据学校基本信息（名称、类型、预算、规模等），自动生成一套完整的校园文化全案建设方案 HTML 单文件，可直接在浏览器打开或发送给客户。

## 包含内容

- **首页 Banner** — 学校名称、预算区间、建设周期、目标定位
- **项目理解** — 学校特点、阶段特征、核心需求
- **四阶九步建设路径图** — Mermaid 流程图，四个阶段颜色区分
- **阶段详情** — 诊断定位 / 理念构建 / 视觉定型 / 空间落地，各含工作内容、预算、交付物
- **方案汇总** — 总览卡片 + 详细预算清单
- **实施建议** — 分阶段策略、预算优化方案、关键成功要素
- **下一步行动** — 建议客户接下来做什么

## 技术栈

- 纯 HTML + CSS + JS，单文件无后端依赖
- Tailwind CSS（CDN）
- Mermaid.js（CDN，流程图渲染）
- Noto Sans SC（Google Fonts）

## 快速开始

### 在 WorkBuddy 中使用

当需要为某所学校生成校园文化方案时，在 WorkBuddy 中描述项目背景，系统会自动调用本 Skill 生成 HTML。

### 本地预览

```bash
# 直接用浏览器打开生成的 HTML 文件
start {文件名}.html
```

## 适用场景

- 新建学校文化体系从零构建
- 已有学校文化升级改造
- 投标文件中的方案附件
- 客户汇报演示
