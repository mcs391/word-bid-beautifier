# proposal-html-generator

全案型方案 HTML 单文件生成工具。

## 功能

根据项目基本信息（名称、类型、预算、周期等），自动生成一套完整的项目方案 HTML 单文件，可直接在浏览器打开或发送给客户。

本工具因校园文化咨询项目实践沉淀，但其设计理念和模板结构**适用于任何行业/类型的方案文档**——校园教育、政企数字化、品牌营销、活动展会等均可使用。

## 包含内容

- **Banner 头部** — 方案名称、类型标签、预算区间、周期
- **项目理解** — 需求分析、现状诊断、核心挑战
- **建设路径图** — Mermaid 流程图，各阶段颜色区分
- **分阶段详情** — 各阶段目标、工作内容、交付物、预算
- **预算汇总** — 结构化预算明细表
- **实施建议** — 关键里程碑、风险提示、关键成功要素
- **下一步行动** — 建议客户接下来的具体步骤

## 技术栈

- 纯 HTML + CSS + JS，单文件无后端依赖
- Tailwind CSS（CDN）
- Mermaid.js（CDN，流程图渲染）
- Noto Sans SC（Google Fonts）

## 快速开始

在 WorkBuddy 中描述项目背景，系统会自动调用本 Skill 生成 HTML 文件。

## 适用场景

- 建设规划类方案（校园文化、数字化转型、智慧城市等）
- 服务提案类（咨询、策划、设计类项目）
- 投标文件附件
- 客户汇报演示
