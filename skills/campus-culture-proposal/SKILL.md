# 校园文化全案建设方案生成器

## 用途

为教育文化咨询项目快速生成专业的校园文化全案建设方案 HTML 页面，适用于客户汇报、招标提案、内部评审等场景。

## 使用时机

用户描述以下意图时触发本 Skill：
- "帮我做一个校园文化全案方案"
- "生成校园文化建设方案HTML"
- "为XX学校做一个文化提案"
- "新建学校文化建设方案"
- "校园文化全案建设"

## 核心资产

模板文件位于 `assets/campus_culture_proposal_template.html`，使用 `{{PLACEHOLDER}}` 格式标记可替换内容。

## 工作流程

### Step 1：收集项目信息

向用户确认以下关键信息（如果用户未主动提供）：

| 信息项 | 说明 | 默认值（可覆盖） |
|--------|------|----------------|
| 学校名称 | 完整名称 | "XX学校" |
| 学校类型 | "初中" / "小学" / "完中" | "初中" |
| 办学层次 | 小学/初中/高中特征描述 | 根据类型自动填充 |
| 预算范围 | 战略+视觉预算（万） | "35-50万" |
| 建设周期 | 总工期 | "6-8个月" |
| 规模 | 班级数量/建筑面积 | 无默认值 |
| 服务方名称 | 显示在页脚 | "问对文化传播" |
| 特殊诉求 | 用户额外说明的需求 | 空 |

> **注意**：如果用户只提供了学校名称和基本描述，直接使用默认值生成初稿，不要求用户填完所有字段。

### Step 2：参数化模板

读取 `assets/campus_culture_proposal_template.html`，将所有 `{{PLACEHOLDER}}` 替换为实际内容。

**必填替换项（无默认值，必须由用户提供）**：
- `{{SCHOOL_NAME}}` — 学校名称

**推荐替换项（用户提供则替换，否则使用默认值）**：
- `{{BUDGET_RANGE}}` — 预算区间，如 "35-50万"
- `{{CONSTRUCTION_PERIOD}}` — 周期，如 "建设周期：6-8个月"
- `{{SCHOOL_SIZE}}` — 班级数量/建筑面积
- `{{SERVICE_PROVIDER}}` — 服务方名称（出现在页脚）

**根据学校类型自动判断的项**：
- `{{SCHOOL_TYPE}}` — "初中" / "小学" / "完中"
- `{{STAGE_FEATURES_TITLE}}` — 阶段特征标题
- `{{STAGE_FEATURES_CONTENT}}` — 阶段特征内容
- `{{CULTURE_STAGE_TITLE}}` — 育人体系特色标题
- `{{CULTURE_STAGE_FEATURES}}` — 育人特色模块

### Step 3：生成 HTML 文件

将替换后的完整 HTML 写入输出目录，建议命名格式：

```
{学校名简称}校园文化全案建设方案.html
```

文件为**单文件纯静态 HTML**，无外部依赖（Tailwind/Mermaid 均通过 CDN 加载），可直接在浏览器打开或通过微信/邮件发送。

### Step 4：向用户呈现

使用 `preview_url` 工具在 IDE 内置浏览器预览，或直接告知用户文件路径。

## 内容结构（默认四阶段）

| 阶段 | 内容 | 预算占比 |
|------|------|---------|
| 第一阶段：诊断定位 | 文化战略工作坊、SWOT分析、使命愿景价值观 | ~10% |
| 第二阶段：理念构建 | 办学理念体系、育人目标体系、教师发展体系 | ~50% |
| 第三阶段：视觉定型 | 核心话语/LOGO/IP、视觉应用系统（34项） | ~30% |
| 第四阶段：空间落地 | 重点空间设计（理念厅/图书馆/走廊等） | 按面积 |

## 预算估算默认值（可按用户实际调整）

```
战略+视觉合计：35-58万
空间设计：30-60万（按实际）
全案总预算参考：65-110万
```

## 常用补充内容

当用户提供以下额外信息时，相应替换模板中的默认值：

- **竞争学校信息** → 替换"差异化定位"模块
- **已有文化基础** → 调整第一阶段诊断重点
- **装修进度** → 调整第四阶段空间实施时间线
- **家长群体特征** → 调整育人体系建议

## 输出质量标准

- 所有价格、人数、面积等数字需精确（保留合理有效数字）
- Mermaid 流程图每个阶段颜色必须区分
- 预算表合计金额必须与各阶段分项之和一致
- 页脚显示服务方名称
- 手机端可正常阅读（响应式布局）
